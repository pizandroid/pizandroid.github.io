Permissions required and why:

Call Filter Alarm Pro requires some authorizations to work. Here is described what they are and why:

  • READ_PHONE_STATE: required for the call filter
  • RECEIVE_SMS: required for the sms filter
  • READ_CONTACTS: required for contacts selection
  • READ_EXTERNAL_STORAGE: required for tone selection
  • RECEIVE_BOOT_COMPLETED: required for restore alarms on phone boot
  • CHANGE_WIFI_STATE: required for wifi managing
  • ACCESS_WIFI_STATE: required for check wifi state
  • ACCESS_NETWORK_STATE: required for check data connection state
  • CHANGE_NETWORK_STATE: required data connection mode managing
  • VIBRATE: required for activate the vibrate mode
  • WRITE_SETTINGS: required for offline state managing
  • CHECK_LICENSE: required for license check
